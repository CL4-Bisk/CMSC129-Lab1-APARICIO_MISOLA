# 🎮 ML o aKO

> A fan-made project inspired by the game, Mobile Legends Bang Bang (MLBB) game, developed using <strong>FERN</strong> or Firebase, Express, React, and Node. The project features a two-screen interactive scenario where the user sets both the user and the opponent's items for better understanding of how one item can have the advantage of the other. This repository demonstrates CRUD operations and an addition to FERN development.
